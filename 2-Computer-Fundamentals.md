**# Module 2 - Computer Fundamentals**



**Overview**

Before entering into securing technology need to understand

how it works  from hardware basics to the cloud.



**What I Learned**



**Inside a Computer**

* &#x20;CPU is responsible for executing instructions and making calculations
* &#x20;RAM is used for temporarily storing information when the computer is on
* &#x20;HDD/SSD stores permanently information on a computer
* &#x20;Motherboard connects all components inside a computer
* &#x20;GPU processes graphics and visual information



**System Privilege Layers**

* Kernel Space - privileged core of OS that provides

&#x20;   unrestricted access to all hardware

* User Space - where regular applications are executed

&#x20;   having limited hardware access

* System calls - how apps request kernel help



**Networking Hardware**

* Router - directs traffic between networks
* Switch- connects devices within a network
* Access Point- provides wireless connectivity



**Cloud Computing**

* &#x20;Cloud stores data on remote servers
* &#x20;Three types- Public, Private, Hybrid
* &#x20;Services- IaaS, PaaS, SaaS



**Summary**

Knowledge of computer hardware and the privilege

layers of systems is important in cybersecurity.

It is easier to recognize weaknesses when you know

how things work.

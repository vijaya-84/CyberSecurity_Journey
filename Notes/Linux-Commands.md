**# Linux Commands Cheat Sheet**



**Navigation**

Command - Purpose - Example 



pwd - Show current folder - pwd 

ls - List files - ls 

cd - Change directory - cd Documents 

mkdir - Create folder - mkdir newfolder 



**File Commands**

cat - Read file - cat readme 

touch - Create file - touch file.md 

rm - Delete file - rm file.md 

rm -rf - Delete folder - rm -rf folder 

cp - Copy file - cp file1 file2 

mv - Move file - mv file folder/ 



**Special Commands**

cat ./- - Read file named "-" 

cat "file name" - Read file with spaces 

find - Search for files 

grep - Search inside files 



**SSH Commands**

ssh user@host -p port - Connect to server 

exit - Disconnect from server 



**Git Commands**

git add . - Add all changes 

git commit -m "msg" - Save with message 

git push - Send to GitHub 

git status - Check changes 

git clone url - Download repo

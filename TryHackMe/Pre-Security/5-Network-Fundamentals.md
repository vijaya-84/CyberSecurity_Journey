**# Module 5- Network Fundamentals**



**Overview**

Learn core concepts of how computers

communicate and types of network weaknesses.



**What I Learned**



**Networking Basics**

* &#x20;Network- devices connected to share data
* &#x20;IP address- unique identifier for each device
* &#x20;MAC address- hardware address of device
* &#x20;Two types of IP- IPv4 and IPv6



**OSI Model- 7 Layers**

* &#x20;Layer 7 Application - HTTP, FTP, DNS
* &#x20;Layer 6 Presentation- encryption, formatting
* &#x20;Layer 5 Session     - managing connections
* &#x20;Layer 4 Transport   - TCP and UDP protocols
* &#x20;Layer 3 Network     - IP addressing, routing
* &#x20;Layer 2 Data Link   - MAC addresses, switches
* &#x20;Layer 1 Physical    - cables, signals, hardware



**TCP vs UDP**

* &#x20;TCP- reliable, checks all data arrives
* &#x20;UDP- fast, does not check delivery
* &#x20;TCP used for- web, email, file transfer
* &#x20;UDP used for- video streaming, gaming, DNS



**Ports and Protocols**

* &#x20;HTTP - port 80
* &#x20;HTTPS- port 443
* &#x20;SSH  - port 22
* &#x20;FTP  - port 21
* &#x20;DNS  - port 53



**Network Weaknesses**

* &#x20;Man in the middle attacks
* &#x20;Packet sniffing
* &#x20;Port scanning
* &#x20;Denial of Service (DoS) attacks



**Summary**

Networking knowledge is the foundation of

cybersecurity. Understanding how data moves

across networks is essential for detecting

and defending against network based attacks.

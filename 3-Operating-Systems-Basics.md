**# Module 3 - Operating Systems Basics**





**Overview**

Understand operating system basics and learn how to use

Windows and Linux through both graphical and command prompt interfaces.



**What I Learned**



**What is OS (Operating System)?**

* It manages both hardware and software resources.
* It acts as an intermediary between the user and hardware.
* Examples – Windows, Linux, macOS



**Windows Basics**

* File system - C:\\ drive structure
* Task Manager - monitor running processes
* Registry - stores system configuration
* Windows Defender - built-in security tool
* User accounts - Administrator vs Standard



**Linux Basics**

* Everything is a file in Linux
* File system starts at root /
* Terminal is primary way to interact
* Users and permissions control access
* Key directories:

&#x20; /home  - user files

&#x20; /etc   - configuration files

&#x20; /var   - log files

&#x20; /tmp   - temporary files



**Command Line Interface**

* &#x20;Faster and more powerful than GUI
* &#x20;Essential for cybersecurity work
* &#x20;Linux commands used daily in SOC roles



**Summary**

Both Windows and Linux are used in real

cybersecurity environments. Linux dominates

servers and security tools while Windows

is common in corporate environments.

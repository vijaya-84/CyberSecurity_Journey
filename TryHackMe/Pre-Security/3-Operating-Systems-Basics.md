**# Module 3- Operating Systems Basics**



**Overview**

Learn fundamentals of operating systems and

how users interact with Windows and Linux

through graphical and command line interfaces.



**What I Learned**



**What is an OS?**

* &#x20;OS manages hardware and software resources
* &#x20;Acts as bridge between user and hardware
* &#x20;Examples- Windows, Linux, macOS



**Windows Basics**

* &#x20;File system- C:\\ drive structure
* &#x20;Task Manager- monitor running processes
* &#x20;Registry- stores system configuration
* &#x20;Windows Defender- builtin security tool
* &#x20;User accounts- Administrator vs Standard



**Linux Basics**

* &#x20;Everything is a file in Linux
* &#x20;File system starts at root /
* &#x20;Terminal is primary way to interact
* &#x20;Users and permissions control access
* &#x20;Key directories:

&#x20; /home - user files

&#x20; /etc  - configuration files

&#x20; /var  - log files

&#x20; /tmp  - temporary files



**Command Line Interface**

* &#x20;Faster and more powerful than GUI
* &#x20;Essential for cybersecurity work
* &#x20;Linux commands used daily in SOC roles



**Summary**

Both Windows and Linux are used in real

cybersecurity environments. Linux dominates

servers and security tools while Windows

is common in corporate environments.

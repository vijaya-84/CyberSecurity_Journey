&#x20;**What I Learned**



&#x20;**DNS in Detail**

* &#x20;DNS translates domain names to IP addresses
* &#x20;Like a phonebook for the internet
* &#x20;DNS Record Types:
* &#x20; A record - maps domain to IP address
* &#x20; CNAME    - maps domain to another domain
* &#x20; MX       - email server address
* &#x20; TXT      - verification information



&#x20;**HTTP in Detail**

* &#x20;HTTP - how browsers talk to web servers
* &#x20;HTTPS - encrypted secure version of HTTP
* &#x20;HTTP Methods:

&#x20; GET    - request data from server

&#x20; POST   - send data to server

&#x20; PUT    - update existing data

&#x20; DELETE - remove data

* &#x20;Status Codes:

&#x20; 200 - Success

&#x20; 301 - Moved permanently

&#x20; 404 - Page not found

&#x20; 500 - Server error



&#x20;**How Websites Work**

* &#x20;HTML - structure and content
* &#x20;CSS  - styling and appearance
* &#x20;JS   - interactive functionality
* &#x20;Web servers send these files to browsers



&#x20;**Full Web Request Process**

* &#x20;User types URL in browser
* &#x20;DNS resolves URL to IP address
* &#x20;Browser sends HTTP GET request
* &#x20;Server sends back HTML/CSS/JS
* &#x20;Browser renders the website

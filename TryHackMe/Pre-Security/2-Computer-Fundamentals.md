**# Module 2- Computer Fundamentals**



**Overview**

Before securing technology you need to understand

how it works- from hardware basics to the cloud.



**What I Learned**



**Inside a Computer**

* &#x20;CPU processes all instructions and calculations
* &#x20;RAM stores temporary data while computer runs
* &#x20;Storage (HDD/SSD) stores permanent data
* &#x20;Motherboard connects all components together
* &#x20;GPU handles graphics and visual processing



**System Privilege Layers**

* &#x20;Kernel Space- privileged core of OS with

&#x20;    unrestricted access to all hardware

* &#x20;User Space- where normal applications run

&#x20;   with restricted hardware access

* &#x20;System calls- how apps request kernel help



**Networking Hardware**

* &#x20;Router- directs traffic between networks
* &#x20;Switch- connects devices within a network
* &#x20;Access Point- provides wireless connectivity



**Cloud Computing**

* &#x20;Cloud stores data on remote servers
* &#x20;Three types- Public, Private, Hybrid
* &#x20;Services- IaaS, PaaS, SaaS



**Summary**

Understanding computer hardware and system

privilege layers is essential for cybersecurity

— knowing how systems work helps identify

where vulnerabilities can exist.
